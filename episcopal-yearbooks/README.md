# Statistics of the Protestant Episcopal Church

These statistics are drawn from the yearbooks of the Protestant Episcopal Church. See this [full description of these yearbooks](http://notebook.lincolnmullen.com/Protestant%20Episcopal%20Church%20yearbooks) for an explanation of the changing titles of the yearbooks and links to all of the sources.

## Files

-   `episcopal-yearbooks-yearly.csv` contains the yearly summaries (as opposed to diocesan or parish records) for the denomination as a whole.
